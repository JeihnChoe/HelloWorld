# Git 명령어

## Git 최초 시작하기

- git config --global user.name "roarroar"
- git config --global user.email "baronjae@naver.com"

## Git 시작하기

- git init
- git add .
- git commit -m "최초 커밋"

## Github 업로드하기

- git remote add origin 주소
- git push origin master

## 최초업로드 끝나고 다시 업로드하는법

- 소스코드를 변경하고
- git add .
- git commit -m "변경내용적고"
- git push origin master
